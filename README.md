
# Blockchain-Secured-Audit-Log

This repository contains **both research and implementation** for the topic:

**Blockchain in Cybersecurity: Research and Implementation of a Small-Scale Blockchain System for Secure Data Storage**

## Structure
- `/research` – Full academic research paper
- `/implementation` – Blockchain system source code

## Features
- Tamper-evident audit log
- SHA-256 hash chaining
- Ed25519 digital signatures
- Attack simulation and verification
